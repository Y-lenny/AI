# 用户服务协议（大纲 · 法务定稿前勿外发）

> 以下为产品级条款结构，需由执业律师结合属地法律、支付模式与类目修订后上线。

## 1. 定义

- 平台、用户、需求方、服务方、任务、里程碑、直播/录像内容。

## 2. 服务性质

- 平台提供 **信息发布、沟通工具与进程记录**，不构成劳动派遣或居间担保（若采用线下付款记录模式，应 **明确平台非付款担保方**）。
- 「去中间商」不表示平台免除 **合法合规与禁止性业务** 的管控义务。

## 3. 账号与实名

- 注册条件、账号安全、禁止转租。
- 实名/资质（分阶段启用）与类目准入。

## 4. 内容与行为

- 禁止欺诈、侵权、淫秽、暴恐、非法数据采集等。
- 对 UGC（含视频、直播）平台有权 **删除、限流、封禁**，详见 [UGC 审核流程](./ugc-moderation-flow.md)。

## 5. 直播与录制

- 单独授权条款：是否录制、用途（纠纷取证/质检）、保留周期、撤回与删除请求入口。
- 未成年人保护：禁止无监护场景下收集未成年人影像（按法规调整）。

## 6. 交易与纠纷（与 MVP 线下记录一致）

- 里程碑与验收规则由双方确认；**线下转账风险自担**。
- 平台可提供 **证据包导出**（任务、会话元数据、交付物哈希等），不保证司法采纳。
- 争议解决：协商 → 平台工单 → 诉讼/仲裁（约定管辖）。

## 7. 责任限制

- 不可抗力、第三方网络与云服务中断。
- **间接损失** 免责上限（依法无效条款除外）。

## 8. 终止

- 用户注销、平台终止服务、数据保留最短政策。

## 9. 联系方式

- 客服、侵权投诉邮箱、响应时限承诺。
